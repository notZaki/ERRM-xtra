Notes on Controls: The 'Maximum Value' text box will set the limits for the colourscale. If it is left empty (default setting) then the largest value in the data will be used. Type 0.1 into that textbox to update the figure, making it resemble the error map for ERRM shown in Figure 1 of the manuscript.

Change the model to 'CERRM' or 'RRM' to see the other two error maps in Figure 1. For the latter, you'll have to change the max value from 0.1 to something higher, e.g. 100, since the RRM has large percent errors.

The purpose of this figure is to show errors for other parameters that were not shown in manuscript. The standard deviation of noise (in mM) and temporal resolution (in seconds) can also be adjusted. Hover the mouse over the error map and the 'Recent Percent Error' box will show the corresponding percent error.
