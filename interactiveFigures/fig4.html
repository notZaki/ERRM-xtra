Model:     Parameter
Sigma:     TRes:     Use as Max Value

To produce maps in manuscript:

  • set 'Model' to either ERRM or CERRM
  • set 'Parameter' to kep
  • set 'Sigma' and 'TRes' to 1
  • enter 0.2 as 'Maximum Value'

    • Options for 'Model' are:

  • Tofts Model (TM)
  • Reference Region Model (RRM)
  • Extended Tofts Model (ETM)
  • Extended Reference Region Model (ERRM)
  • Constrained Extended Reference Region Model (CERRM)

    • The 'sigma' option is just an index, and covers the noise range of stdDev = 0:0.01:0.1 mM.
    So 1 = noiseless, 2 = 0.01 mM noise, 3 = 0.02 mM noise, etc.

    The 'Maximum Value' option sets the limits of the colourscale.
    Any regions with an absolute percent error greater than the max value will be set to black.
    If no value is entered, then the absolute max will be used by default
